# Rsbuild Project

## Setup

Install the dependencies:

```bash
pnpm install
```

## Get Started

Start the dev server:

```bash
pnpm dev
```

Build the app for production:

```bash
pnpm build
```

Preview the production build locally:

```bash
pnpm preview
```

# Playwright E2E Tests

Run Playwright tests:

```sh
bun run test:e2e
```

Open Playwright UI:

```sh
bun run test:e2e:dev
```
